# Bitmasks

**Bitmask and state management nodes.**

| Node             | Description                                                                    |
| ---------------- | ------------------------------------------------------------------------------ |
| **Write States** | Write point states as bitmask attributes from filter evaluation.               |
| **Point State**  | Filter-driven point flagging with bitwise operations (Set, OR, AND, XOR, NOT). |
