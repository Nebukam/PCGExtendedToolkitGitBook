# Attribute

**Filters that test point attribute values.**

| Node                        | Description                                                         |
| --------------------------- | ------------------------------------------------------------------- |
| **Numeric Compare**         | Compare numeric attribute values (equal, greater, less, etc.)       |
| **Numeric Compare Nearest** | Compare against nearest point's attribute value                     |
| **Numeric Self Compare**    | Compare a point's own attributes against each other                 |
| **String Compare**          | Compare string attribute values (exact, contains, starts/ends with) |
| **String Self Compare**     | Compare a point's own string attributes                             |
| **Boolean Compare**         | Test boolean attribute values                                       |
| **Gameplay Tags**           | Match Unreal Gameplay Tags                                          |
| **Value Hash**              | Test computed hash values                                           |
| **Within Range**            | Test if numeric value falls within min/max range                    |
