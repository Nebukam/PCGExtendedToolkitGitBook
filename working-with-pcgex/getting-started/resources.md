# Resources

PCGEx builds on top of Unreal's PCG framework. If you're already comfortable with PCG, skip ahead to the concepts. If not, these resources will get you up to speed.

{% hint style="info" %}
## PCGEx extends PCG&#x20;

**it doesn't replace it.** \
A solid understanding of vanilla PCG makes everything in PCGEx click faster.
{% endhint %}

#### Procedural Generation Primer

If you're new to procedural generation as a discipline:

* [**Procgen Primer for PCG**](https://jeanpaulsoftware.com/2025/02/25/first-we-make-manhattan-then-we-make-berlin/) — An intro to procedural generation by Ed Boucher

#### Unreal PCG Basics

Recommended resources for learning Unreal's PCG framework:

* [**Adrien Logut Tutorials**](https://www.youtube.com/@adrien_logut/videos) — A must-watch if you're serious about PCG
* [**Starting out with PCG**](https://www.youtube.com/watch?v=Uc8TGLIeNT0) — Getting started guide
* [**PCG Overview**](https://dev.epicgames.com/documentation/en-us/unreal-engine/procedural-content-generation-overview) — Epic's official documentation
* [**PCG Essentials**](https://www.youtube.com/playlist?list=PLYrS3D9rJwiS5Mhp5HZFxXnj-_NmfilrW) — Video playlist covering fundamentals

#### Feature Discovery & Deep Dives

Once you have the basics, these channels explore specific techniques and workflows:

* [**Procedural Minds**](https://www.youtube.com/@procedural_minds) — PCG tutorials and procedural tool development
  * [PCGEx-focused playlist](https://www.youtube.com/watch?v=wnlU3ZKLrJ4\&list=PLokuMVKY46wjdTePd55zSB4RK0Gy2kyPX)
* [**Aziel Art**](https://www.youtube.com/@azielarts) — Practical skills for digital artists working with procedural tools

{% hint style="success" %}
There's no single "right way" to use PCG. Many approaches lead to the same result — some with better performance or maintainability than others. Explore, experiment, and find what works for your project.
{% endhint %}
