# Transform

**Nodes that move, orient, and relax point positions.** Transform Points applies arbitrary position/rotation/scale changes, Best Match Axis aligns points to a reference direction, Move Pivot shifts the transform origin, Lloyd Relax distributes points evenly in 2D or 3D, Flat Projection collapses points onto a plane, and Tensors Transform applies tensor field directions to point orientations.
