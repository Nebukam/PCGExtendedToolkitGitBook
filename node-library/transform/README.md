# Spatial

**Point-level spatial operations that work on any data — no cluster or path context required.** Generating new points from existing ones, fusing overlapping points, transforming positions and orientations, and analyzing spatial properties like collocation and bounds.

| Section                           | Contents                                                                      |

| [modify](modify/ "mention")       | Reshape point sets — fusing, bounds conversion, bin packing                   |
| [transform](transform/ "mention") | Move and orient points — transforms, relaxation, projection, tensor-driven    |
| [analyze](analyze/ "mention")     | Inspect spatial properties — collocation, normalization, bounds decomposition |
