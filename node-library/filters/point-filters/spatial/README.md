# Spatial

**Filters based on spatial relationships and containment.**

| Node                 | Description                                    |
| -------------------- | ---------------------------------------------- |
| **Distance Filter**  | Test distance between points or to a reference |
| **Bounds Filter**    | Test bounding box containment                  |
| **Inclusion Filter** | Test if point lies inside a region             |
| **Time Filter**      | Time-based filtering                           |
