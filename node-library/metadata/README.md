# Metadata

**Attribute manipulation — reading, writing, transforming, and managing point metadata.** These nodes work with the data on points rather than with points themselves: generating new attributes, modifying existing ones, computing statistics, and managing bitmask state.

### Sections

| Section  | Contents                                                              |
| -------- | --------------------------------------------------------------------- |
| Generate | Create new attributes — indices, GUIDs, constants, tuples, actor data |
| Modify   | Transform existing attributes — blending, remapping, cleanup          |
| Analyze  | Inspect attribute data — hashing, statistics, reduction               |
| Bitmasks | Bitwise operations on integer attributes — set, clear, test, combine  |
