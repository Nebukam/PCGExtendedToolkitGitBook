# Path Statistics

Configures statistics tracking for pathfinding operations.

### Overview

This settings block enables tracking of how often vertices and edges are used across all discovered paths. By writing use counts to attributes, you can identify high-traffic nodes and edges in the cluster â€” useful for visualizing path density, finding bottlenecks, or weighting subsequent operations based on path popularity.

### How It Works

1. **Run Pathfinding**: Multiple paths are computed through the cluster
2. **Count Usage**: Track how many paths pass through each vertex and edge
3. **Write Attributes**: Store use counts on the original cluster data
4. **Analyze Results**: Use counts for visualization or downstream processing
