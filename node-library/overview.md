# Overview

**This is the reference section.** Every PCGEx node is documented here — inputs, outputs, settings, behavior. If you need to understand what a node does and how to configure it, this is where you look.

The library is organized by what you're working with, not by source module. Find the system you're using, then drill into the operation type.
